# run-butane-with-github-actions

The file [example.fcc](example.fcc) is automatically transpiled into the ignition file [example.ign](https://eriksjolund.github.io/run-butane-with-github-actions/example.ign) by
[butane](https://github.com/coreos/butane) in a [Github actions workflow](.github/workflows/butane.yml). The file [files/file1.txt.](files/file1.txt) is included into the Ignition file.
